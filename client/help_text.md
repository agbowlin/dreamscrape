# DreamScrape

DreamScrape is an R&D tool to assist in developing web scraping projects.

## Commands

DreamScrape is controlled by a set of commands (steps) that you provide
to navigate and manipulate web pages.

### Navigation

- Url
- WaitFor

### Manipulation

- SendText
- SendKey
- Click

### Scraping

- ScrapeText
- ScrapeValue
- ScrapeArray
- ScrapeTable

### Other

- Debug
- Snapshot
